# Changelog

## v1.6.0 - ThingsBoard Stable Bridge

- Added WiFi non-blocking reconnect.
- Added MQTT connection to ThingsBoard.
- Added telemetry upload every 30 seconds.
- Added force first telemetry sync after MQTT connection.
- Added device attributes: firmware, IP, MAC and reset reason.
- Kept OLED, sensors, LEDs, buzzer and buttons running during network loss.

## v1.5.x - Local Comfort Device

- Added OLED local dashboard.
- Added DHT22 temperature and humidity reading.
- Added heat index calculation.
- Added LDR relative light sensor.
- Added focus proxy score.
- Added 3-color LED alert logic.
- Added buzzer alert profiles.
- Added navigation and advice buttons.

## v1.0.0 - Initial Prototype

- Initial ESP32 local sensor prototype.
